# Security Policy

## Supported Versions

Currently, only the latest release of the Macro Intelligence Platform receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project (e.g., dependency vulnerabilities, code injection risks, or issues with local cache handling), please do not disclose it publicly.

Instead, please open a private security advisory on GitHub or email the maintainer directly. We will review the vulnerability and release a patch as soon as possible.
